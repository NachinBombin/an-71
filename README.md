# an-71

Weaponless Garry's Mod AN-71 flyover addon.

## Features
- Atmospheric AN-71 support plane entity
- No weapons at all
- NPC-triggered support call flow
- Flight settings in Q-menu
- Uses model path `models/an71/an71.mdl`

## Notes
- Sister addon to the AC-130 addon, but stripped to flight/spawn/menu systems only.
